# BusReservationSystem
Bus Reservation System 📌 Project Description

The Bus Reservation System is a console-based Java application developed using Java, MySQL, and JDBC. It allows users to book bus tickets, view available buses, check booking details, and cancel reservations.

The system follows a layered architecture (Model, DAO, Service, UI) to ensure proper separation of concerns and efficient database interaction.

⚙️ Technologies Used Java MySQL JDBC ✨ Features View available buses Book bus tickets View booking details Cancel reservations Search booking by passenger name Seat availability check (prevents overbooking) 🗄️ Database

Users can interact with the system through a menu-driven interface. The application connects to the MySQL database using JDBC to store and retrieve booking information dynamically.

🎯 Objective

To develop a simple and efficient bus reservation system that demonstrates database connectivity, CRUD operations, and structured Java application design.
